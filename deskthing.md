# Desk Thing
Hey there! This is a intermediate project that will use display graphics and WiFi!   
  
Note that this is a intermediate project, step by step instructions will not be provided! Google and Youtube are your best friends during this journey. 
    
** Due to [spotify's updated API](https://developer.spotify.com/blog/2026-02-06-update-on-developer-access-and-platform-security) usage, the spotify desk thing is still available to those who have spotify premium. **  

Because of this, the tutorial will be more of project guidelines to guide you in the right direction!

### How this works

You will be creating a display that sits on your desk, hence the name desk thing! This device can do anything from display the weather, to your notifications, to a calendar.  
You will be required to use certain components and certain aspects of code. 

### Software Requirements
- The project must interact with the internet! Use the WiFi.h library or any other method you would like
- Thats about it for the software, as long as the code works with the final product, you're good. 
- You may ask: "how are we supposed to create code when we dont have components?" You can you simulation websites such as [circuitio](https://www.circuito.io/), [tinkercad](https://www.tinkercad.com/), and [wokwi](https://wokwi.com/)!

### Hardware Requirements

Your project must include the following
- User input (whether that be buttons, touch sensors, etc.)
- Microcontroller with WiFi (ESP32, RPI Pico, etc.)
- A screen (ST7735, OLED, Epaper)
- A case (CAD it! Use any platform you would like)
- You can choose to hand solder your project, or create a pcb for it!

### Case Requirements

Please refer to the [spotify display tutorial](https://github.com/Dongathan-Jong/SpotifyDisplay/blob/main/Tutorial.md) for the case preview, you may use the tutorial to get started, but you **MUST** personalize it and change the layout, otherwise **your project will not be approved**!

### Submission

To submit, make sure your repo has the following: 
- Code
- Case
- PCB (if needed)
- Hardware Components
- BOM
- README! Make sure this is descriptive, let us know if you are planning to hand wire / solder!

Good luck!
